The YumJam One-Off Specials plugin extends Woocommerce allow support for products that are one-off or bespoke

== Description ==

The YumJam One-Off Specials plugin extends Woocommerce allow support for products that are one-off or bespoke, this extra functionality allows these product listings to remain in the shop after sale and replaces the add to basket button with an easy to configure popup inquiry form.

Below are some of its features:

 - Seamless Woocommerce integration
 - Compatible with all themes that have Woocommerce template support
 - Setting embedded within Woocommerce->Settings->Products
 - Popup Enquiry form support for all major form plugins
 - Html or Shortcode pop content
 - Re-sizeable popup window

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/yumjam-one-off-special-products` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin

== Frequently Asked Questions ==

= What does One-Off Specials plugin Do? =

he YumJam One-Off Specials plugin extends Woocommerce allow support for products that are one-off or bespoke

= Can I customise the woocommerce out-of-stock message? =

Yes, from the plugin setting page you can override the out-of-stock message for selected products.

= Can I customise the popup window content? =

Yes, you can populate the pop windows with shortcode based content and it is compatible with every shortcode based form plugin e.g. Gravity Forms, Contact Form 7, etc..
